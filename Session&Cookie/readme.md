用来判断客户端是谁的工具。

session:服务器机制，服务器在客户端请求后回复一个唯一的sessionId给客户端，这样在下一次http请求时服务器就知道客户端是谁了。

cookie:客户端机制，客户端存储本地的用户数据（用户操作历史信息，登录信息）来判断。
